## 核心框架速查

### 王煜全：荷塘效应
池塘荷叶每天翻倍，第30天铺满。第27天只有12.5%。
**赢家在第27天就已决定，大多数人等到铺满才恍然大悟。**

### Zero：631原则
- 六成 = 剧本（决定下限）
- 三成 = 画面张力（决定上限）
- 一成 = 整体质量（密度 + 逻辑自洽）

### Zero：仿拆量三步法
1. 仿（像素级复刻爆款，练提示词，累计100小时）
2. 拆（逐帧拆解100组爆款，找核心爆点）
3. 量（融会贯通后量产）

### 刘小排：AI编程六级阶梯
| 级别 | 形态 | 特征 |
|------|------|------|
| 1 玩具 | v0/Bolt | 网页生成demo |
| 2 工具 | Cursor | 做出商业产品 |
| 3 副驾 | VS Code+插件 | 复杂项目 |
| 4 命令行 | Claude Code | 睡前布置任务 |
| 5 实习生 | Claude Code进阶 | 业务语言沟通 |
| 6 全职员工 | OpenAI Codex | 自主找活干 |

**分水岭：前三级是「AI帮你做」，后三级是「AI替你做」**

### 老华：好产品9条支线
1. 产品差异化 2. 供应链壁垒 3. 产品定价
4. 市场规模 5. 商业模式 6. 找到对的人
7. 晋升路径 8. 可持续迭代组织 9. 企业文化/底线

### 王大仙：Agent五步部署法
1. 数据清理与权限搭建
2. 选配模型（预设预算，无焦虑才敢放开用）
3. 赋予灵魂（语音描述你是谁、想让Agent干啥）
4. 学技能Skill（把工作流变成U盘）
5. 放入生产环境

### 刘智行：赛道选择六维模型
满足5项以上 = 优质赛道：
流量维度 / 变现维度 / 内容维度 / 竞争维度 / 合规维度 / 个人维度

### 郭东超：三段式内容创作法
1. 场景描述（具体画面，让用户一秒带入）
2. 痛点共情（低姿态，我因为什么痛点做了什么）
3. 引出方案（产品是「我的解决方案」，不是「你应该买的东西」）
