## 第二部分：能学到什么（道法术器拆解）

### 嘉宾一：王煜全（投资人/趋势观察者）

#### 道（刻进骨子里）

**刻意选择胜过自然选择**

- 自然选择成功率很低——大多数人只是跟着大势走，和别人想一样的事，做一样的改进
- 刻意选择是先看清未来，再倒推现在该做什么
- 他的原话：「如果你先做的答案就是巨大的成就」——没人做过的事，才有最大的机会
- 对比：做淘宝的时候你再做淘宝没机会了，做微信的时候你再做微信没机会了

**服务平权是这个时代最大的底层机会**

- 工业革命实现了物质平权（穷人和富人喝同一瓶可乐）
- AI时代要实现的是服务平权——富人有家教、有理财顾问、有家庭医生，穷人没有
- 美国最富11%的人比最穷人群平均寿命长10岁——因为健康服务不平等
- 未来的Agent就是每个人的「参谋部」，不再是一个参谋，而是一群AI协作给你建议
- 做服务普惠的人会比服务富人的人更成功——「把服务能力导入穷人手里，保证你比服务富人的人更有钱」

**从用户角度而非对立面提供服务**

- 传统商业是站在用户对面收钱（美团、淘宝模式）
- 未来是站在用户这一侧，做用户的数字分身
- 乔布斯不是「了解用户需求」，而是「为自己造了iPhone，碰巧全世界都喜欢」
- Agent不是你的员工，不是你的客户，是「你自己的一个更强版本」

#### 法（很少变的方法论）

**技术领先→文化领先→应用领先**

- 无线互联网时代中国领先（抖音、电商、短视频出海）
- AI时代，前三年是大语言模型（美国领先），未来三年是大行为模型（中国领先）
- 因为中国无线互联网最发达，行为数据最丰富，做行为分析最有优势
- 关键区分：大行动模型(large action model)只看一次性行为，大行为模型(large behavior model)能连续预测行为

**荷塘效应——趋势在最后几天才被大众看见**

- 荷叶每天翻倍，第30天铺满池塘，第29天才50%，第28天25%
- 当市场看起来「稀稀落落」的时候，懂规律的人就知道——铺满只差最后几天
- 「我们不是预测未来，我们是看清荷塘一定会被铺满」

#### 术

- 智能硬件围绕手机做「使用体验优化」，缩短行为闭环（录音豆、夸克眼镜）
- AI出海：用中国的应用经验带AI出海，而不是技术出海
- 未来不是用一个AI，而是用多个AI讨论后给你综合结果（「参谋部」模式）

#### 器

- 手机是边缘侧算力中心，短期内不会被替代
- 各种智能硬件（录音豆、手环等）贴在手机上缩短操作路径

### 嘉宾二：Zero（AI视频创作者）

#### 道

**普通人翻盘靠的是抓住窗口期，不是天赋**

- Zero本人的经历：27岁之前当服务员、工地搬砖、电话营销，连Excel都不会
- 2025年之前一个视频都没做过，连B站是什么都不知道
- 但半年内拿下4亿+播放量、300万订阅、月入万刀
- 核心不是天赋，是「这个机会点就是Sora/C-320，目前仍然属于中国的实践窗口」
- 他的原话：「论谁的天资更低，论谁的资历更差，这件事没有人比我更差」

**人类内容的本质是情感演绎**

- 技术壁垒会不断被拉平——每次工具更新，所有创作者回到同一起跑线
- 唯一不会被拉平的是「对人类情感的理解和演绎」
- 假特效、假魔法、假大片都不是本质，只有情感共鸣才是内容的终极护城河
- 「人类最喜欢看的核心内核，就是情感演绎的最大化」

#### 法（631原则——短视频的底层密码）

**一个视频能不能成为爆款，6成看剧本，3成看画面张力，1成看整体质量**

- **剧本（6成）**：好剧本的唯一标准——发布1-3天内能拿到1000万播放量。不用猜，去找已验证的爆款剧本，逐帧拆解它的核心爆点
- **画面张力（3成）**：不是空泛的「好看」，而是人类眼球7要素——死亡、暴力、前进、钱、西方、阴暗、异常。这7种东西从古至今人类就喜欢看
- **整体质量（1成）**：不是画面清不清晰，而是「逻辑自洽」。一个片段如果逻辑不通（比如换个发型老公就认不出来），观众潜意识会觉得不对

**密度与质量——对抗观众的潜意识走神**

- 人类对一个片段的基础理解时长是1-1.5秒
- 超过这个时间，观众就有「潜思考时间」——开始想别的事，然后划走
- 解决方案：在每个片段结束前的0.5秒加入一个「变化」（表情变化、视角切换），占据观众全部注意力
- 密度 = 在有限播放时间内保持最大内容输出
- 这需要极其精细的提示词控制——精确到每一秒发生什么

#### 术（普通人过门槛的三步法）

**仿 → 拆 → 量**

1. **仿**：找到近期爆款，一模一样地复刻。不是「差不多」，是视角一样、机位一样、表情一样、什么都一样。目的不是做爆款，是练习「掌控提示词的能力」
2. **拆**：拆100组爆款剧本。不是看一遍就完了，要逐帧拆解，横向对比，找到核心爆点。「不是得到野，不是这样，你只会越拆越低」——要拆的是核心爆点的规律
3. **量**：累积至少1小时的AI视频制作时长。Zero自己累积了数千小时。「没有捷径，仿就是最高效的方法，拆就是最快的捷径」

**掌控时间单位的内容量**

- C-320一次只能出15秒，同一段提示词在不同时间段产生的效果截然不同
- 4秒太紧凑，11秒太松弛，6-8秒可能刚好
- 要精确控制每一秒出现什么——「事无巨细地给它描述清楚」

#### 器

- Sora / C-320：目前最强的AI视频生成工具，「��格万象皆可AI」
- YouTube作为变现平台

### 嘉宾三：波波（GEO/AI搜索优化）

#### 道

**流量入口正在从「搜索→推荐」转向「AI对话」**

- 用户决策链路中，AI已经开始替代人做信息查找、筛选、评估
- 体验比传统搜索好10倍——「用搓衣板洗过衣服再用洗衣机，你还回得去吗？」
- 这不是趋势，是正在发生的事：春节期间豆包AI聊天量激增，父母都在用
- 抖音每天日搜索PV 40亿+，且持续增长

**AI平台不生产内容，只有算法和算力**

- AI回答的质量取决于它能获取到什么信息源
- 谁提供了AI平台信赖的信息，谁就占据了AI时代的流量入口
- 「只要我们洞察了用户需求，提供好的内容让AI先看到、理解、最后作为答案回答给用户，就完成了获客」

#### 法（GEO方法论）

**成为行业的信息枢纽**

- 案例：九数云在官网发了30万+篇文章，1000万+篇内容全被搜索引擎和AI平台收录
- 不是只推自己，而是做横向对比——「抖音数据分析工具对比深度测评」
- 看起来客观中立，但竞品的缺点多写一些，自己的优点多写一些
- 这不是投机取巧，而是占据「行业信息权威」的生态位

**移动端才是关键**

- 移动端AI平台的日活和月活远超PC端
- 315后豆包等平台把移动端回答大量导向抖音商城和抖音内容
- 真正有需求的场景（找餐厅、订外卖、查产品）大多发生在移动端

**AI喜欢「已经有人总结好的」内容**

- AI也要消耗算力，它更倾向于直接引用已经结构化好的文章，而非去海量数据中实时分析
- 美团几千万SPU，全量接入元宝后每天只有1万次查询——因为结构化数据AI处理不过来
- 解决方案：中间层翻译——用自然语言文章把结构化数据包装好，AI就能直接引用

#### 术

- 美团春节GEO案例：用30万预算占领「蜜雪冰城在哪点最便宜」等关键词
- 天鹅到家案例：在抖音做搜索SEO，意外获得GEO曝光，800万搜索曝光带来7000万交易额
- 律师个人IP：做北京离婚律师推荐等关键词，1小时内内容就出现在AI回答中
- 投流验证：一个传统工厂老板（50多岁女性），通过豆包看到推荐，3天下单

#### 器

- 波波自研的GEO数据分析工具（品牌AI排名、引用源追踪）
- 抖音搜索 + AI摘要（移动端）
- 豆包、千问等AI平台

### 嘉宾四：钱塘江鲤（AI Agent / 小龙虾应用）

#### 道

**从「方案」到「结果」——AI的价值跃迁**

- 以前的AI（ChatGPT、豆包）给你方案——写篇文章、做个分析
- 但执行链条还是需要人：复制粘贴、打开后台、上传图片、定时发布
- Agent（小龙虾/OpenClaw）给你结果——你只需要说一次需求，它自动完成全部链条
- 「答案是方案，但结果才是最有价值的」——老板要的是第二天桌上的调研报告，不是一个任务分解

**AI越用越聪明，人越用越轻松**

- Agent会记录你的偏好、习惯、工作流
- 越用越懂你——最终变成真正的「贾维斯」式个人助理
- 它甚至会做你没想到的事——比如自主去看竞品的路线图，给你产品迭代建议

#### 法

**SOP越详细，Agent越稳定**

- 很多人说Agent不稳定——执行10次只有3-5次达到预期
- 根本原因：指令不够清楚
- 一个生产级的SOP可能需要5000字甚至1万字
- 两个关键：① 用最强的模型（Claude 4.6、GPT-5.4）② 把流程写到「再也加不了一个字」的程度

**一次调试，持续运行**

- 和Agent的正确交互模式：先用对话调通一个流程 → 然后设为定时任务 → 每天自动执行
- 就像带新员工：第一周手把手教，第二周开始他自己跑

#### 术

- **选题自动化**：让Agent每天从公众号、小红书、Twitter抓取热点，分析后推送到飞书
- **招聘自动化**：Agent在招聘平台批量打招呼→收集简历→按岗位匹配打分→推送结果+面试建议
- **竞品监控**：监控竞品官网变化、社群讨论、新品发布→整理成报告推送
- **舆情分析**：监控用户对产品的正面/负面评价，总结期待方向

#### 器

- OpenClaw / 小龙虾（开源Agent框架，GitHub星标历史最高，超过React）
- Claude / GPT作为底层模型
- 飞书作为信息汇集终端
- 微信、Slack等通讯工具作为Agent入口

### 嘉宾五：郭东超（小程序创业者）

#### 道

**产品好的话，用户会自发帮你传播**

- 郭东超发现自己的小程序突然流量暴涨——原来是抖音用户在评论区自发推荐，其中一条评论点赞过万
- 「产品好的话真的会带来自传播」——不需要你主动推，用户会帮你推
- 这也是他做产品的核心原则：先解决自己的痛点，做出自己想用的产品

**过去所有经历都是在往「人力账户」存钱**

- 他参加了无数航海（小红书、抖音、视频号直播带货），大部分失败了
- 但正是这些失败教会了他怎么写笔记、怎么投流、怎么分销、怎么做直播
- 「不管成功还是失败，都是往人力账户存钱。等你找到适合你的机会，这些能力就会帮你快速起飞」
- 他的方法论不是临时学的，是6年积累的串联

#### 法

**场景描述 → 痛点共情 → 引出方案（内容创作三步法）**

- 不是「推广我的产品」，而是「分享我的经历」
- 开头写一个特别具体的画面——「每次开会轮到我发言，脑子里明明有想法，一上台就一片空白」
- 用户看到后自然带入自己的经历，产生共情
- 然后引出解决方案——不是硬推产品，而是「我用了什么方法解决了这个痛点」
- 用户的评论验证了效果：「和我一模一样」

**多维组合矩阵——永远不缺内容方向**

- 一个痛点下面有无数种人群（职场人、学生、宝妈）、无数种场景（开会、面试、约会）、无数种情绪触发时刻
- 把人群x场景x痛点x情绪x时刻组合起来，算出来有74万种+笔记方向
- 永远不需要模仿别人，用AI直接帮你从这些维度创作

#### 术

**冷启动（0→100用户）**

- 小红书验证：一周做MVP → 直接上线 → 用「场景+痛点+方案」写原创笔记
- 前10天零流量零转化——因为只是在模仿别人的笔记形式
- 调整为原创场景描述后，第30天突破100付费用户

**投流不是买单，是买速度和确定性**

- 新手投流心态错误：投100块没出单就赶紧关闭
- 正确心态：投流是投资，花200块买到数据反馈——知道哪篇笔记点击高、哪篇转化好
- 然后用数据反馈去优化笔记，最终ROI达到2.2（花12万，卖26万）

**增长突破（8000→2万+用户）**

- 从小红书跳到抖音——小红书是启动器，抖音是放大器
- 在小红书4个月获得1万用户，在抖音不到1个月翻倍
- 关键动作：怼脸拍短视频→直播→达人分销→素人打造→抓溢出流量

**代理策略**

- 给代理70%高佣——让他们把这件事当成自己的事
- 筛选有互联网经验的宝妈，而非随便找人
- 一个宝妈代理第二个月就帮他带来1500+付费用户

#### 器

- 微信小程序 + 抖音小程序（双平台）
- Claude（AI辅助创作笔记）
- 小红书投流后台（数据分析）

### 嘉宾六：老华（连续创业者/投资者）

#### 道

**创业是「创」不是「投」**

- 米克3万起步，14年持续经营；马场老火锅200万起步，3年130家直营店
- 创业核心是从零到一创造价值，不是砸钱
- 「很多人创业投1000万——那叫投资不叫创业」

**底层逻辑只有一句话，但需要无数支线支撑**

- 做出有需求的好产品——道理谁都懂
- 但背后至少有9��支线：产品差异化、供应链、定价、市场规模、商业模式、找到对的人、晋升路径、可持续迭代的组织、企业文化
- 「如果这9个问题你都做的很好，你的项目不可能不成功」——但现实是很难全做好
- 竞争对手来学马场，回去为什么都不成功？「因为支线不止一条，是一个系统工程」

**投资就是买入生意的一部分——标的比价格重要100倍**

- 不要被「计算企业未来现金流折现」带偏——变量太多，算不准
- 核心是判断：这是不是一个好生意？
- 好生意的标准：高毛利、低持续投入、有复购（茅台 vs 京东方）
- 「宁愿在价格上犯错，也不要在标的上犯错」
- 6000点买100倍PE的茅台，拿到今天至少赚30-50倍——价格贵了又怎样？

#### 法

**差异化+供应链规模=护城河**

- 马场只做一款红锅——其他品牌不敢只做辣锅
- 只做一款锅→月采购量1000万+→供应链规模优势→别人进不来
- 锅底48元贴着成本卖（市场平均68-80）——用低价锅底引流，靠其他菜品赚钱
- 只做150平小店型→砍掉甜品区→降低模型重量→可快速复制

**做减法比做加法更重要**

- 砍掉甜品（腾出厨房空间）、砍掉鸳鸯锅（只做红锅）、砍掉花哨装修（省钱）
- 结果反而更好——简单的装修反而被模仿，只卖红锅反而形成差异化
- 不做团购、不投广告、不打折——「如果是真的打六折，那他一定是亏钱的」
- 总部只有五六个人管3000人的团队——极致精简

**创业者应该是好的资金分配者**

- 创业、投资、跨界本质是一件事：资金分配
- 好的创业者必须懂投资——你要见过好公司，才知道自己公司好不好
- 马场零负债，每年利润买优质股权（比如买了茅台）→ 投资分红可能超过经营利润
- 「花无百日红」——企业不一定能做20年，但优质股权的复利可以帮你穿越周期

#### 术

**用985管培生替代传统餐饮人**

- 马场店长80%以上是985/211毕业生
- 传统餐饮人思路固化——「在我们的模型下就是错的」
- 宁要白纸，因为学习能力强，1-2年成长速度极快
- 很多创新（效率提升、流程优化）是管培生自己在工作中总结出来的

**供给侧稀缺 = 流量密码**

- 老华2020年B站实盘视频爆火——因为之前没人把这个资金规模的实盘放到网上
- 「在短视频内容里，供给侧稀缺的时候，传播度会非常非常大」
- 米克也是——当年最早做外贸在线教育，切中的是刚需（外贸人必须学英文网站推广）

#### 器

- 没有提到具体AI工具——老华坦言「AI跟我好像没什么关系」
- 核心工具是：财务报表（但他说自己不怎么看）、企查查（用来实时跟踪新东方在线销售额）
